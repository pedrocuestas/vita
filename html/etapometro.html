Etapómetro

Elija poblaciones de inicio y final de ruta. Puede arrastrar estos iconos sobre el mapa para modificar el inicio o final.
Luego, accione este icono para calcular las etapas de la ruta planificada.

Otras opciones:
  • Ver el perfil de la ruta
  • Modificar preferencias para cada etapa
  • Resetear o borrar ruta planificada
Tras unos segundos, mostrará en esta misma ventana una tabla con los datos de cada tramo y etapa:
  • Distancia de cada tramo y etapa (uno o varios tramos)
  • Tiempo estimado de cada tramo y etapa
  • Desnivel acumulado positivo
  • Índice de dificultad física comarativo:
    • Nivel alto
    • Nivel medio-alto
    • Nivel medio
    • Nivel medio-bajo
    • Nivel bajo
    • Nivel muy bajo
  • Dificultad técnica
    • Dificultad alta
    • Dificultad media
    • Dificultad baja
  • Valoración de cada tramo
  • Tipo de terreno:
    • Carretera/asfalto (Km)
    • Pista/camino terrizo (Km)
    • Sendero (Km)
